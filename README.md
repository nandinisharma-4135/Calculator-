# Tkinter Calculator

Developed a desktop-based Calculator application using Python and the Tkinter library for the Graphical User Interface (GUI).

## Key Features & Learning:
* Built an interactive desktop GUI using **Tkinter** widgets (Buttons, Entry labels).
* Implemented core mathematical logic and real-time expression evaluation using **Python**.
* Handled layout management efficiently using Tkinter's grid system.
* Focused on clean code structure and basic error handling for mathematical operations.
